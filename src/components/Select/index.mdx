## API - Props
- `theme` 三种主题：`basic`(默认)、`calendar`(日历)、`search`(搜索)
- `renderOption` 选择框选项渲染
- `placeholder` 占位符
- `searchPlaceholder` 搜索下拉菜单占位符
- `onSearch` 文本框值变化时回调
- `onFocus` 搜索框获得焦点时回调
- `onBlur` 搜索框失去焦点时回调
- `onChange` 选择改变的回调
- `onDelete` 删除多选标签时回调
- `onSelect` 选择选项时的回调
