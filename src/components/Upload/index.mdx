## API - Props
- `style` 样式style属性
- `className` 类名
- `theme` 三种主题：`click`(默认)、`card`、`drag`
- `hintMsg` 提示文本内容，可选
- `showHint` 是否展示提示文本，默认展示
- `preview` 是否展示预览，默认不展示
